## 1、灾情指令
````
1.1 地图屏-应急指挥-属地资源-专业队伍-下发指令
    对应后端接口
    POST /kircp/incidentCommand/distributeWithBase64File
1.2 地图屏-应急指挥-周边资源-增援力量和专业队伍-下发指令
    对应后端接口
    POST /kircp/incidentCommand/distributeWithBase64File
1.3 业务屏-应急指挥-力量调度-一键调度 （会生成若干条指令） 备注:全国账号登陆，选择一条地震灾情
    对应后端接口
    POST /kircp/forceSchedule/startDispatch
````
